# Mohamed Portfolio Website

Modern, responsive portfolio website built with React, Tailwind CSS, and Framer Motion.

## Features

- Dark + light theme toggle
- Smooth animations (fade-in, hover, scroll-triggered motion)
- Fully responsive layout for mobile and desktop
- Sections: Hero, About, Skills, Projects, Experience, Education, Contact
- Downloadable CV button
- Admin Dashboard for dynamic content management

## Run Locally

1. Open a terminal in this folder:

```bash
cd C:\Users\yk779\Desktop\mohamed-portfolio
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open the URL shown in terminal (usually `http://localhost:5173`).

## Build for Production

```bash
npm run build
npm run preview
```

## One-Step Deploy

### Vercel

```bash
npm run deploy:vercel
```

### Netlify

```bash
npm run deploy:netlify
```

On first run, each CLI will ask you to authenticate and link the project.
